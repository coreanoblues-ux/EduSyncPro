🎯 목표
학생 등록/수정 화면의 모든 <Select>에서 빈 value("")가 렌더링되지 않게 하고,
placeholder는 <Select.Item>이 아니라 <Select.Value placeholder="...">로만 처리하라.

1) 문제 지점 검색
- 전역에서 아래 패턴을 모두 찾고 목록을 출력하라:
  rg -n "<Select\.Item" src server
  rg -n "value=\"\"" src server
  rg -n "value=\{.*\|\| *''\}" src server
  rg -n "defaultValue=\"\"" src server
  rg -n "onValueChange" src

2) 공통 수정 규칙(모든 Select에 적용)
- ❌ 금지: <Select.Item value="">, value={... || ''}, defaultValue=""
- ✅ 허용:
  <Select.Trigger>
    <Select.Value placeholder="선택하세요" />
  </Select.Trigger>
  <Select.Content>
    {options
      .filter(o => o?.id != null && String(o.id).length > 0)
      .map(o => (
        <Select.Item key={o.id} value={String(o.id)}>
          {o.name}
        </Select.Item>
      ))}
  </Select.Content>

- 컨트롤 값은 반드시 undefined 또는 문자열. 절대 "" 사용 금지:
  // react-hook-form Controller 예시
  <Controller
    control={form.control}
    name="classId"
    render={({ field }) => (
      <Select
        value={field.value ?? undefined}              // "" 대신 undefined
        onValueChange={(v) => field.onChange(v)}      // v는 string
      >
        <Select.Trigger>
          <Select.Value placeholder="반을 선택하세요" />
        </Select.Trigger>
        <Select.Content> ...위 map... </Select.Content>
      </Select>
    )}
  />

- 수정(편집) 폼 진입 시 초기화:
  form.reset({
    classId: existing?.classId != null ? String(existing.classId) : undefined,
    ...기타필드
  });

- 제출 직전 숫자 필드 변환:
  const payload = {
    ...values,
    classId: values.classId ? Number(values.classId) : null,
    tuition: values.tuition ? Number(values.tuition) : 0
  };

3) 빠른 안전장치(옵션 미도착/로딩 시)
- 로딩 중에는 Item을 렌더링하지 말 것:
  {options?.length > 0 ? options.map(...) : null}

4) 반자동 치환 제안
- 다음 코드 패턴을 일괄 치환하라(미리 diff 출력 후 적용):
  value={option?.id || ''}   →  value={String(option.id)}
  value=""                    →  (삭제; placeholder는 <Select.Value>로 대체)
  defaultValue=""             →  defaultValue={undefined}
  value={formValue || ''}     →  value={formValue ?? undefined}

5) 테스트
- 학생 등록/수정 화면에서 모든 Select를 열고 선택/해제해도 런타임 오버레이가 더 이상 뜨지 않아야 한다.
- 값 미선택 상태에서는 placeholder만 보이고, 선택 후에는 문자열 value가 유지되어야 한다.
